# Harmattan

Real-time air quality monitoring platform for Ghana. Built for Tech Hub Africa Hackathon 2026.

## Stack
- Astro 6 + React 19 + Tailwind CSS 4
- Leaflet maps with OpenStreetMap tiles
- WAQI API for real-time air quality data
- Deployed on Cloudflare Pages

## Tracks
- Pollution Control
